# Techvruk Landing Page Concept

This project contains a responsive, visually rich landing page concept for **Techvruk**, focused on first-time visitors and clear brand communication.

## Purpose

The page highlights Techvruk's value proposition:

- Bridging the gap between academic learning and industry technologies
- Upskilling through training programs
- Encouraging innovation through contests and awards
- Building a collaborative engineering and technology community

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

No build tools or external npm packages are required.

## Files

- `index.html` - Landing page structure and content
- `styles.css` - Visual design, layout, responsiveness, and animations
- `script.js` - Scroll reveal effects, stat counters, and CTA form interaction

## How to Run

1. Open `index.html` directly in your browser.

That is enough to run this project locally.

## Design Highlights

- Strong hero section with clear messaging and CTAs
- Impact metrics section with animated counters
- Program pillars (training, contests, awards)
- Journey/timeline showing skill-to-opportunity flow
- Community and movement-focused sections
- Mobile-first responsive behavior

## Notes

- This implementation keeps the existing environment intact.
- No Node version changes were made.
- No dependency installation or upgrade is needed.
